How to use Arc Dictionary Extraction

Steps to success - Instructions


  1. Click the 'Browse' button and select a single PDF file
    • Please do not select any other file type as they are not supported.
    • Must be an OCR'ed file
    • For best results the document should have consistent style and content
  2. The selected file will display in the corresponding label

  1. In the selected PDF find a page to begin extracting words
  2. Enter the PDF page number, not the document's page number

  1. In the selected PDF find a page to end the extraction of words
    • Must be consistent format throughout the page range
    • End search page may be the same as the start search page
  2. Enter the PDF page number, not the document's page number

  1. Within the page range select a keyword to enter
    • A good keyword has related text that spans two lines
    • End search page may be the same as the start search page

  1. Within the page range if there is a general header/footer select the box accordingly
    • This includes document title/page number footers or index headers

  1. If input is valid a table with results will appear below with features for the user to continue editing or export
